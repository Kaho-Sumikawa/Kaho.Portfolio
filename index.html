My Axis

誰かの頭に残るものを、作り続けたい。

01

主体性

与えられた課題より、自分が「面白い」と思えたことのほうが寝食を忘れて没頭できます。その主体性を活かしながら、アイデアを形にすることが得意です。

02

創造

粘土作品のSNS投稿への反応をきっかけに、3Dモデリング・ゲーム開発へと独学で走り続けました。0から考えて手を動かすことが、何より好きです。

03

余韻

作りたいのは、終わったあとも誰かの頭に残るもの。フロントエンドでもゲームでも、「あの体験、よかったな」と思い出してもらえる制作をしていきたいです。

Skills

HTML / CSS

プログラミングスクールやUdemyを通して、HTML/CSSの基礎を学びました。TailwindCSSやBootstrapを使用したコーディングも経験しています。

JavaScript

DOM操作やイベント処理などを用いた、動きのあるWebページ制作に取り組んでいます。

React

プログラミングスクールとUdemyでReactを学習しました。 コンポーネント設計やstateを使った状態管理など、 フロントエンド開発の基礎を身につけています。

TypeScript

プログラミングスクールとUdemyでTypeScriptを学習しました。 型を意識したコードを書くことで、保守性や安全性を意識した実装を心がけています。

Node.js / Express

学内ハッカソンで開発したWebアプリ「知り合いコレクション」にて、 Node.js・Expressを用いたバックエンド設計とルーティング処理を担当しました。 画面・ルーター・データベースの繋がりを意識した設計を経験しています。

MySQL / SQLite

プログラミングスクールでMySQLを使ったデータベース設計を学習しました。 学内ハッカソンではSQLiteを用いて、外部キー制約によるテーブル間の 紐付け設計も実践しています。

C# / Unity

Udemyを通じてC#の基礎を学び、Unityでミニゲームを開発しています。 人型キャラクターのモデリング・リギングにも挑戦し、実際にUnity上で 動かしてアニメーションさせる制作も経験しました。 ミニゲーム「こいのぼりゲーム」をunityroomに公開しています。 将来的にはSteamでの公開を目指しています。

Blender

基本操作をYouTubeで学び、ローポリモデリングを実践しています。 『爆走！近大マグロ』の制作では魚類の3Dモデリングを担当し、 実際にUnity上で動かす制作も経験しました。
